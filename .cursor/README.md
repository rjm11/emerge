### Cursor project config for EMERGE

- This directory provides Cursor-native guidance mapped from `CLAUDE.md` and `CLAUDE REFERENCE/`.
- Rules here do not duplicate the originals; they point to them and define how to use them inside Cursor.

### How to use

- Start tasks with: “Use EMERGE agents …” or select a specific agent (see `rules/agents.md`).
- Include reference paths in prompts when coding: `CLAUDE.md`, `CLAUDE REFERENCE/**`, `README.md`, `REPOSITORY_STRUCTURE.md`, and files you’ll edit.
- For mechanics or game logic, consult `resources/wiki-repo/**` first.

### Files

- `rules/agents.md`: EMERGE agent flow and mappings for Cursor.
- `rules/architecture.md`: Event-driven architecture essentials.
- `rules/context.md`: Where to find authoritative references.
- `rules/testing.md`: Testing, coverage, and debug guidance.
- `rules/style.md`: Style, SemVer, changelog and manifest practices.
